# Pricing And Cache Rates

Last checked: May 6, 2026.

`tokenuse` calculates cost from local usage files and a pricing snapshot. It does not call provider billing APIs during ingestion. The embedded snapshot is generated from LiteLLM pricing data, with small local overrides where official provider pricing publishes cache rates that are missing or stale upstream.

## Conclusion

Claude cache reads being around 10% of input is right. Codex/OpenAI is not broadly 50% anymore for current GPT-5.x and Codex models: current GPT-5.4, GPT-5.5, and GPT-5.x-Codex cache reads are 10% of input. `codex-mini-latest` is 25%. Older OpenAI rows such as `gpt-4o` can still be 50%.

## Implementation Notes

- `src/pricing/snapshot.json` is generated by `cargo run -- --refresh-prices`; do not hand-edit it.
- `cursor-auto` is a real local pricing row, not an alias to Sonnet.
- `anthropic-auto` resolves to the latest Sonnet row, and `openai-auto` resolves to GPT-5.
- The UI's `Cache` column remains observed cache-hit behavior. The new `Cache Rate` column is the pricing multiplier for cached input relative to normal input.
- Session call detail shows both token buckets and per-model cache read/write price rates.

## Parser Caveats

- Claude Code exposes input, output, cache-write, and cache-read buckets directly, so tokenuse can price cache behavior from source data.
- Codex/OpenAI reports cached input inside total input. The parser subtracts cached input before pricing and then prices the cached portion as cache read.
- Gemini reports cached tokens inside input, so the Gemini parser follows the same subtract-then-price pattern.
- Cursor local files do not provide a reliable cache breakdown. `cursor-auto` pricing is correct, but observed cache-read tokens remain 0 until Cursor exposes them in local source data.
- Copilot local transcripts are still estimated and do not expose reliable cache buckets. GitHub's June 1, 2026 AI Credits billing includes cached tokens, but tokenuse cannot reconstruct that billing exactly from local files today.
